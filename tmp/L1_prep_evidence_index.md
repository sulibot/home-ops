## Steps

1. List all 17 Level 1 practices.
2. Map each practice to one or more existing files in `level_1`.
3. Add operational proof beyond templates where required.
4. Mark evidence status as `missing`, `draft`, `current`, or `ready`.
5. Add gap notes for missing or stale evidence.
