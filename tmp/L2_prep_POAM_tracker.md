# Level 2 POA&M Tracker

## Purpose

Track deficiencies, exceptions, and remediation work that must be managed before a Level 2 assessment.

## Scope

- documented control gaps
- missing evidence
- implementation failures
- policy or procedure deficiencies

## Steps

1. Record every deficiency found during documentation review, technical validation, or mock assessment.
2. Link the deficiency to the affected practice or domain.
3. Assign an owner and target date.
4. Attach closure evidence once fixed.
5. Review the tracker at least weekly during readiness work.

## Expected Outputs

- active POA&M
- prioritized remediation list
- closure audit trail

## Evidence or Artifacts Created

- remediation tracker
- closure evidence references

## Definition of Done

- Every material deficiency has a tracked remediation entry.
- Closed entries have linked proof of resolution.
